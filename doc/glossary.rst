.. _glossary:

Glossary
========

.. glossary::

  loader
    A loader is a program that loads data into an Open Spending site.
    We maintain several loaders with Open Spending, provide a generic
    loader that can load csv files you upload through a form in an
    Open Spending site and provide help to
    :doc:`write custom loaders <loaders>`

  mongodb query spec
    A dictionary with a query for mongodb like `{'name': 'society'}` or 
    `{'dataset.name': 'cra'}`. Complex queries are described (with
    javascript examples) in the
    `Mongodb Advanced Query Docs
    <http://www.mongodb.org/display/DOCS/Advanced+Queries>`_.
